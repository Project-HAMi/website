---title: Allocate Vastai Device---This example shows how to request a single Vastai device in a plain Kubernetes Pod.The Pod simply runs a long‑running container image provided by Vastaitech and asks for one `vastaitech.com/va` device through the `resources.limits` section.You can use this as a starting point and adjust the image and resource limits to fit your own workloads.```yamlapiVersion: v1kind: Podmetadata:  name: vastai-podspec:  containers:    - name: test      image: harbor.vastaitech.com/ai_deliver/vllm_vacc:VVI-25.12.SP2      command: ["sleep", "infinity"]      resources:        limits:          vastaitech.com/va: 1```
