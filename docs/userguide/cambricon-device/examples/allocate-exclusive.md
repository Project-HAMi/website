---title: Allocate exclusive device---To allocate a whole cambricon device, you need to only assign `cambricon.com/vmlu` without other fields.```yamlapiVersion: apps/v1kind: Deploymentmetadata:  name: binpack-1  labels:    app: binpack-1spec:  replicas: 1  selector:    matchLabels:      app: binpack-1  template:    metadata:      labels:        app: binpack-1    spec:      containers:        - name: c-1          image: ubuntu:18.04          command: ["sleep"]          args: ["100000"]          resources:            limits:              cambricon.com/vmlu: "1" # allocates a whole MLU```
