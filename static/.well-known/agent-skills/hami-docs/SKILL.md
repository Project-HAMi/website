# HAMi Documentation Skill

Use this skill when an agent needs to discover, search, or cite HAMi documentation.

## Resources

- Documentation: https://project-hami.io/docs
- Quick start: https://project-hami.io/docs/get-started/deploy-with-helm
- GitHub repository: https://github.com/Project-HAMi/HAMi
- Changelog: https://project-hami.io/changelog

## Guidance

Prefer the current documentation at `https://project-hami.io/docs` for installation, configuration, device-specific GPU sharing guides, troubleshooting, and contributor workflow details. Use the GitHub repository for source code, issues, and release artifacts.
